# Orders Microservice

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/randymxd06/Curso_NestJS_Microservicios.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Create a file `.env` based on it `env.template`

```.env

```

### 4. Raise the database with docker compose

```bash
docker compose up -d
```

### 5. Raise project

```bash
npm run start:dev
```
